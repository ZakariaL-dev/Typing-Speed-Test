## 🌟 Overview
**Typing Speed Test** is an interactive web application designed to measure and improve your typing efficiency. Built with **Vanilla JavaScript**, it calculates your speed and accuracy as you type, providing an engaging way to sharpen your keyboarding skills.

## ✨ Key Features
* **⚡ Live WPM Calculation:** Watch your Words Per Minute (WPM) update dynamically.
* **🎯 Accuracy Tracking:** Real-time feedback on your typing precision and error count.
* **⏱️ Countdown Timer:** Race against the clock to see how many words you can finish.
* **📱 Responsive Design:** Clean, centered UI that works across all screen sizes.
* **🔄 Quick Restart:** Instantly reset the test to try and beat your high score.

## 🚀 Installation
Follow these steps to get a local copy up and running:

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/ZakariaL-dev/Typing-Speed-Test.git](https://github.com/ZakariaL-dev/Typing-Speed-Test.git)
    ```
2.  **Navigate to the Project Directory**
    ```bash
    cd Typing-Speed-Test
    ```
3.  **Launch the App**
    Simply open the `index.html` file in your preferred browser:
    ```bash
    open index.html
    ```

## 📖 Usage
1.  Launch the application and click the **Start** button or begin typing in the text area.
2.  Follow the highlighted text and type as fast as you can.
3.  The timer will start automatically upon your first keystroke.
4.  Once finished, review your final **WPM** and **Accuracy** percentage.
